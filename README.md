# 🐾 PetShop

A modern **full-stack pet marketplace** built with **Next.js, Express.js, MongoDB, and JavaScript**. PetShop allows users to discover and purchase their favorite pets while providing pet shop owners with a powerful dashboard to manage pet listings, inventory, and orders.

---

## 🌐 Live Demo

* **Live Site:*https://pet-shop-one-phi.vercel.app* 
* **Frontend Repository:** https://github.com/mdallamaiqbal/PetShop
* **Backend Repository:** https://github.com/mdallamaiqbal/PetShop-server

---

## 📖 Overview

PetShop is designed to simplify the process of buying and managing pets online. Customers can browse available pets, view detailed information, and place orders through an intuitive interface. Pet shop owners have access to a secure dashboard where they can efficiently manage pet listings and monitor store activities.

---

## ✨ Features

### 👤 User Features

* Browse available pets by category
* View detailed pet information
* Search and filter pets
* Add pets to cart
* Secure user authentication
* Place and manage orders
* Responsive design for all devices

### 🛠️ Pet Shop Owner Features

* Secure dashboard
* Add new pet listings
* Update pet information
* Delete pet listings
* Manage inventory
